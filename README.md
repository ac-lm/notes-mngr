__Aplicación Spring que simula ser un bloc de notas con funcionalidad para _crear_, _editar_, _archivar_, y _eliminar_.__  

_Cómo correr la app en tu computadora_  
1. Descargar e instalar Docker Desktop (https://www.docker.com/products/docker-desktop/)
2. Ejecutar Docker Desktop
3. Clonar (o descargar) el proyecto    
4. Abrir la carpeta del proyecto en el explorador de archivos
5. Abrir una consola de comandos dentro de la carpeta (en Windows: click derecho -> "Abrir en terminal")
6. Ejecutar en la consola _docker-compose up -d_
7. Acceder a http://localhost:5500/ en el navegador
